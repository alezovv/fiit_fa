# Темы к экзамену по Фундаментальным Алгоритмам 2025/2026.

1. Платформа .NET. CLR (Common Language Runtime) и JIT-компиляция. Процесс компиляции в C#.
2. Управление памятью в .NET. Стек (Stack) vs Куча (Heap). Различия ссылочных типов и значимых типов. Понятие Boxing/Unboxing и их влияние на производительность.
3. Сборка мусора (Garbage Collector). Поколения объектов, фазы работы GC.
4. Generics в C#. Механизм реализации дженериков в CLR. Ковариантность и контравариантность.
5. Объектно-ориентированное программирование на C#. Классы. Инкапсуляция. Наследование. Статический и динамический полиморфизм. Перегрузка методов и операторов.
6. Итераторы и интерфейсы `IEnumerable`/`IEnumerator`. Механизм реализации итераторов в языке C#. 
7. Ассоциативные контейнеры. Деревья двоичного поиска (BST). Алгоритмы вставки элемента по ключу в BST, удаления элемента по ключу из BST, поиска элемента по ключу в BST, поиска элементов в диапазоне ключей в BST. Префиксный, инфиксный и постфиксный обходы деревьев двоичного поиска.
8. Ассоциативные контейнеры. АВЛ-дерево. Балансировка АВЛ-дерева при вставке элемента по ключу, при удалении элемента по ключу.
9. Ассоциативные контейнеры. Красно-чёрное дерево. Балансировка красно-чёрного дерева при вставке элемента по ключу, при удалении элемента по ключу.
10. Ассоциативные контейнеры. Splay-дерево. Балансировка splay-дерева при вставке элемента по ключу, при удалении элемента по ключу, при поиске элемента по ключу.
11. Ассоциативные контейнеры. Декартово дерево. Балансировка Декартового дерева при вставке элемента по ключу, при удалении элемента по ключу, при поиске элемента по ключу.
12. Арифметические алгоритмы. Элементарные операции над длинными целыми числами: сложение чисел, вычитание чисел, умножение в столбик, целочисленное деление в столбик, взятие остатка от деления в столбик, поразрядные операции (конъюнкция, дизъюнкция, инверсия, сложение по модулю 2), операции сравнения, операции отношения порядка; алгоритмические сложности описанных операций. Дополнительный код числа, алгоритм получения дополнительного кода. Организация хранения длинного вещественного числа впамяти компьютера. 
13. Алгоритмы с многочленами от одной переменной. Арифметика многократной точности. Сложение и вычитание целых чисел. Умножение и деление целых чисел. Тривиальные алгоритмы
14. Арифметические операции над элементами множеств различной природы. Понятие группы, коммутативной группы. Нейтральный элемент группы. Обратный элемент группы. Понятие кольца. Аксиомы кольца. Кольца вычетов по модулю 𝑛. Обратимые элементы кольца вычетов. Понятие поля. Поле вычетов по модулю 𝑛.
15. Многочлены от одной переменной. Кольца многочленов.
16. Классический алгоритм умножения многочленов (умножение в «столбик»), анализ его сложности. Сравнение с алгоритмом умножения целых чисел.
17. Алгоритм умножения Карацубы, анализ его сложности.
18. Примитивные корни из единицы и их свойства. Дискретное преобразование Фурье и его свойства. Обратное дискретное преобразование Фурье.
19.  Алгоритм быстрого преобразования Фурье. Алгоритм быстрого умножения многочленов на основе быстрого преобразования Фурье. Алгоритм Шёнхаге — Штрассена.
